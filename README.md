Огляд житлового будівництва в Мельбурні  

Цей набір даних було зібрано з загальнодоступних результатів, які публікуються щотижня на Domain.com.au. Набір даних включає адресу, тип нерухомості, передмістя, спосіб продажу, кімнати, ціну, агента з нерухомості, дату продажу та відстань від центрального ділового району.  

При аналізі даних були виконані наступні кроки:  
  - Перейменування стовпців
  - Перетворення типу даних: стовпець date був перетворений на тип datetime.
  - Обробка нульових значень та аномалій: нульові значення та нелогічні значення були замінені на NaN.
  - Перевірка на дублікати: Дублікати в даних не були виявлені.
  - Аналіз розподілу цін: розподіл цін нерухомості є скошеним вправо, більшість будинків коштують у діапазоні 500 тис. – 1.5 млн австралійських доларів. Логарифмічне перетворення цін дає більш симетричний розподіл.
  - Аналіз ціни за типом житла: будинки (тип 'h') мають найвищу медіанну ціну та найбільший розкид, тоді як квартири (тип 'u') є найдешевшими з найменшим розкидом цін. Таунхауси (тип 't') знаходяться посередині.
  - Виявлення кореляції з ціною: найсильніше позитивно корельовані з ціною такі ознаки, як rooms, bedroom, bathroom. Це вказує на те, що кількість кімнат, спалень та ванних кімнат є ключовими факторами вартості нерухомості. landsize та propertycount мають дуже слабку кореляцію з ціною.
  - Створення нових ознак: були створені нові ознаки, такі як sale_year, sale_month, age та suburb_freq. Також були додані бінарні ознаки для відстеження наявності даних (has_building_area, has_year_built, has_landsize). Деякі оригінальні стовпці були видалені через низьку кореляцію або надлишковість.
  - Використання моделей: серед розглянутих моделей (DummyRegressor, LinearRegression, RandomForestRegressor) найкращі результати показала модель RandomForestRegressor.
    
Таким чином, проведений аналіз даних про нерухомість Мельбурна дозволив виявити ключові фактори, що впливають на ціноутворення, а також розробити прогнозну модель. Після ретельного очищення даних, перетворення типів та обробки пропущених значень, було встановлено, що кількість кімнат (rooms), спалень (bedroom) та ванних кімнат (bathroom) є найважливішими ознаками, що позитивно корелюють з ціною. Також було помічено, що будинки (h) є найдорожчими, тоді як квартири (u) — найдоступнішими.

Завдяки інженерії ознак, яка включала вилучення року та місяця продажу, розрахунок віку будівлі та додавання бінарних індикаторів наявності даних, вдалося покращити якість даних для моделювання.

Серед протестованих моделей, RandomForestRegressor показала найкращі результати, досягнувши показника R2 = 0.7679 з MAE 189,110.20 та RMSE 303,657.62. Це свідчить про те, що модель може пояснити близько 77% дисперсії ціни та забезпечує досить точні прогнози, перевершуючи базові та лінійні моделі.

Таким чином, цей проєкт успішно демонструє, як дані про нерухомість можуть бути використані для отримання рішень та побудови ефективних прогнозних моделей, що може бути корисним для оцінки вартості нерухомості на ринку Мельбурна.
